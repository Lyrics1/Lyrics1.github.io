yoman 脚手架

2017-11-24

初步构建

安装yoman

npm install -g yo

查看版本： yo –version

在这里构建的是react 项目：

  1. generator-react-webpack

npm install -g generator-react-webpack

查看安装版本命令：
npm ls -g –depth=1 2>/dev/null | grep generator-

npm ls -g //列出npm 全局安装的包

>表示重定向

–depth 数状结构1 层

1表示 标准输出 2 表示标准错误输出

>2>/dev/null 表示有错误消息，将错误消息重定位：错误消息过滤输出

grep generator- :检索generator 开头的

使用

生成项目
yo react-webpack 文件名

根据相应项目进行选择所安装的依赖
