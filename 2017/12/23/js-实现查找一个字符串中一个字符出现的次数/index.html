js-实现查找一个字符串中一个字符出现的次数

2017-12-23

找出一个字符中只出现过一次的字符：

var T =”abbcdfgttt”;
var A = […new Set(T)];
var B =[];
for(var i=0;i<A.length;i++){
var n = (T.split(A[i])).length-1;
if(n==1){
B.push(A[i]);
}
}

var T =”abshcj”;
var A = Array.from (T);// 将T转化为一个数组；
var B = new Set(A);//得到一个去重的数组B ，必须是数组，但是得到的是Set 对象，所以使用数组元素取法娶不到；

所以又将 B 转化为数组：
B = Array.from(B);

然后就这样；

var A = […new Set(T)];
这样直接就得到一个去重的可以取到的数组；
