数据库1

2017-02-08

Myaql 默认端口号： 3306；

Mysql 超级用户： root;

  1. 创建数据库

    CREATE DDATABASE 数据库名； (语句结束标志 英文分号 ；)

    以下数据库名以name代替；

    1. 数据库的编码查看及其指定编码

    查看： SHOW CREATE DATABASE name;

    指定编码 ：CREATE DATABASE t1 CHARACTER SET UTF8; (假设指定编码为utf-8)

    1. 数据库的修改 : ALTER DATABASE t1 CHARACTER SET GBK;

    2. 删除数据库 : DROP DATABASE t1;

  2. 打开数据库 : USE t1;

检验打开数据库是否是t1; SELECT DATABASE();

  1. 创建表： CREATE TABLE
